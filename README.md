# Chatbot-Tkinter 🤖

A simple GUI-based chatbot built using Python and Tkinter.

## Features
- Interactive chat interface with purple theme
- Responds to greetings, farewells, thanks
- Answers project-related queries
- Built with Tkinter for cross-platform GUI
- Runs on desktop and Pydroid 3

## How to Run
```bash
python chatbot.py
